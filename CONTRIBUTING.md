# Contributing to Prism VST Plugin

Thank you for considering contributing! To submit changes:

1. Fork the repository on GitHub and clone your fork.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with clear messages.
4. Push the branch to your fork and open a pull request.

Please ensure the project builds and tests pass before submitting:

```bash
mkdir build
cd build
cmake ..
make
./PluginHost
```

By contributing you agree that your code will be released under the MIT License.
